【开发描述】
windows 下的 bison/flex 开发模板，包含tcc编译器，项目在 win64 下能直接运行
总大小小于 5M ，且不需要其他任何依赖。不过因为使用的是 tcc 编译器所以只能在纯 c 语法环境中运行
如果需要 cpp 语法之类的处理，请使用 g++ ，同时需要修改 run.bat 中编译器的部分即可。

【使用方式】：
直接双击 "run.bat" 或 直接在命令行直接执行 "run.bat" 即可使用 c.l 和 c.y 生成一个exe文件。
具体命令行请看 run.bat 文件里面的命令行是怎么处理的即可。该 bat 会删除 flex 和 bison 生成的全部中间文件。
如果你想要保留中间生成的文件则在命令行中使用 "run.bat test" 会保留中间文件：y.tab.c y.tab.h y.output lex.yy.c

【一点描述】：
flex 主要处理 c.l 文件
bison 主要处理的是 c.y 文件
tcc 主要处理的是用上述两个文件生成的中间文件编译成 a.exe 。
这里最后的 a.exe 就是最终生成的我们需要的编译器。
这个 a.exe 在当前模板 c.y 和 c.l 编译下，功能只能解析简单的整数的加减乘除和括号的语法。
双击运行 a.exe 即可测试。
